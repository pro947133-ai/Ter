Команды Termux по категориям

Поиск и фильтрация по ключевым словам и сценариям. Команды сгруппированы по системным задачам, работе с пакетами, сетью и автоматизацией.

Собрали быстрые сценарии

Комбинируйте команды в зависимости от задачи — от настройки окружения до анализа сети. Каждая подборка кликабельна: просто скопируйте и адаптируйте под себя.

Быстрый старт проекта

  • pkg install git nodejs
  • git clone <repo>
  • npm install && npm run dev

Подходит для веб-проектов и CLI-инструментов. Добавьте tmux, чтобы не терять сессию.

Аудит сети

  • pkg install nmap net-tools
  • ip addr
  • nmap -sV <подсеть>

Используйте в связке с tmux и логированием для плановых проверок инфраструктуры.

Резервное копирование

  • pkg install rclone
  • tar -czf backup.tgz ~/project
  • rclone copy backup.tgz remote:termux

Настройте расписание через crontab, чтобы копии создавались автоматически.

Как быстрее ориентироваться в Termux

Используйте горячие клавиши и расширения оболочки, чтобы ускорить работу. Комбинируйте команды из справочника с автоматизацией.

Горячие клавиши терминала

Ctrl + A/E для перемещения по строке, Ctrl + L для очистки экрана, Ctrl + R — поиск по истории.

Сеансы с tmux

Создавайте несколько окон и панелей, сохраняйте сессии даже после закрытия Termux.

Ускоренный ввод

Используйте alias для типовых команд, подключайте автодополнение и плагины оболочки.

Сборка проектов

Установите clang, cmake, make и запускайте полноценные пайплайны CI прямо на устройстве.

Совет

Создавайте собственные коллекции команд в файле ~/notes/termux.md или через task/just. Так вы сможете запускать сложные сценарии одной командой.
